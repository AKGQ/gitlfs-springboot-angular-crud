1. git init
2. git lfs install
3. git status
4. git add . //Guardar cambios, meter cosas en la caja
5. git lfs track "CRUD_Notas"
6. git status
7. git commit -m "Primer cambio" //Cerrar caja y poner cinta con texto de contenido de envio
8. Crear repositorio en github.com, al cual se va a subir y seguir los 3 comandos

__________BRANCHES__________
git branch //See branches
git branch branchname //Create branch

__________PULL__________
git pull //Actualizar todas las ramas de github
git pull origin branchname //Traer de github a una rama en especifico

__________CLONE__________
Copy repository url (example: https://github.com/AKGQ/tutorial2.git)
git clone "https://github.com/AKGQ/tutorial2.git" (With parentesis)
